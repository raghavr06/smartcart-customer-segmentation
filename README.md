# SmartCart Customer Segmentation

## Problem Statement
Segment customers into meaningful groups based on purchasing behavior and demographics to enable targeted marketing strategies.

---

## Dataset
- Contains customer demographics, purchase history, and engagement metrics  
- Includes features like income, spending patterns, product categories, and visit frequency  

---

## Tech Stack
- Python  
- pandas, NumPy  
- scikit-learn  
- Matplotlib  

---

## Approach

### 1. Data Preprocessing
- Performed feature engineering to transform raw attributes into more meaningful features
- Handled outliers to improve clustering quality 
- Scaled numerical features  

### 2. Clustering Techniques
- K-Means Clustering  
- Agglomerative Clustering  

### 3. Cluster Evaluation
- Elbow Method used to determine optimal number of clusters  

### 4. Visualization
- PCA-based visualization for cluster interpretation  

---

## Results
- Identified distinct customer segments  
- Clusters showed clear separation based on spending behavior  

---

## Business Impact
- Enables targeted marketing campaigns  
- Helps identify high-value customers  
- Assists in improving customer retention strategies
